# 🍑 安装
## 📦 环境要求

❗ **重要**：本组件库需要以下最低版本环境：
- ⚡ Vue.js ≥ 3.3.0
- 🟢 Node.js ≥ 18.x

## 📥 安装方式

选择一个你喜欢的包管理器

::: code-group

```shell [npm]
npm install peach-x
```

```shell [yarn]
yarn add peach-x
```


```shell [pnpm]
pnpm add peach-x
```
:::
